    Estrutura Básica do HTML

    (VOLTAR)

    Nessa aula foi explicado como é contituido a estrutura de um HTML,Exemplo:

  • index.html : Página inicial do HTML.
  • html : Tudo que estiver dentro desta TAG é denominado HTML.
  • head : Todas as informações que são pré carregadas precisam estar entre essa TAG como o "title".
  • body : Tudo que estiver entre essa TAG ira aparecer de fato para o usúario, ou seja é o corpo do projeto.
    • OBS: todas as TAGS utilizadas deverão estar entre as devidas pontuações.

    Falando Sobre TAGS

    (VOLTAR)

    A TAG nada mais é que códigos que executam alguma função ou ação para o HTML, alguns exeplos de TAGS são:

  • i : A fonte ficara em italico.
  • hr : Adiciona uma linha divisora entre uma informação e a outra.
  • p : Nada mais é que uma caixa de texto.
  • strong : Altera a fonte para negrito.

    • Atributos das TAGS

    (VOLTAR)

    São propiedades derivadas das TAGS, exemplo:

  • id: Esse atributo é derivado da TAG "strong", é utilizado para identificar o elemento do HTML no javascript.
  • style: É utilizado para aplicar comandos do CSS direto no HTML, como por exeplo a coloração da fonte.
  • class: Ele ira padronizar todos os Titulos principais.
  • type: Atributo utilizado na TAG "input", sendo possivel criar caixa de texto ou númericos .
  • src: Atributo utilizado na TAG "img", nada mais é que o caminho da imagem que será exibida.

    • Textos

    (VOLTAR)

    Nesta aula foi apresentado alguns modelos de TAGS para texto, algumas delas são:

  • h1: TAG utilizada para nomear/representar Títulos.
  • h2: O navegador entende que os demais atributos numericos após a letra "H" exeto o número 1, serão Subtítulos.
  • u: Essa TAG é utilizada para Sublinhar palavras ou frases.
  • mark: Utilizado para Grifar o texto.

    • Listas Ordenadas e Não Ordenadas

    (VOLTAR)

    As listas ordenadas são usadas para criar uma sequencia em formato de topicos para o usúario, um exemplo dela é:

  • li: Ele ira ele ira listar os itens por topicos com uma bolinha .

    • Links

    (VOLTAR)

    Nessa aula foi demonstrado algumas possibilidades de navegação onde há a possibilidade de alternar as telas, ou clicar em algum comando onde o usuario é levado para outros locais pré definidos no HTML, exemplo:

  • a href: Utilizando esta TAG, você ideentifica/indica o caminho para onde o usuário sera levado CLIQUE AQUI.
  • targuet: Exitem três tipos de atributos para o "targuet":
    1. _self: Usuário ao clicar no LINK o navegador não irá brir outra aba.
    2. _blank: Ao selelcionar essa opção o navegador abrirá uma nova aba.
    3. title: Ao estacionar o mouse sobre o link, aparecerá uma coixa de texto abaixo do cursor, sendo possivel a modificação do texto.
